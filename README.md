# Marketing Campaign Effectiveness Analysis

## Project Overview
This repository contains a professional-grade data visualization created using Seaborn to analyze marketing campaign effectiveness.

## Visualization Details
The scatterplot visualizes the relationship between:
- **X-axis**: Marketing Spend (in thousands of dollars)
- **Y-axis**: Conversion Rate (%)
- **Bubble Size**: Customer Engagement Score (1-10 scale)
- **Color**: Campaign Type (Digital, Social Media, Email, TV)

## Business Insights
The visualization helps identify:
1. Optimal marketing spend levels
2. Most effective campaign types
3. Engagement-to-conversion relationships
